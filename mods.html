MODS

Magisk

A powerful tool for rooting Android devices, enabling modifications without altering system partitions.

KernelSU

A kernel-level rooting framework designed to simplify device customization and management.

LSPosed

A lightweight and efficient framework for managing Xposed modules on modern Android systems.

Iconify

An app to customize and personalize system icons, creating a unique Android experience.

HyperCeiler

A performance-enhancing tool for optimizing and managing system resources effectively.

Viper4Android

A powerful audio modification tool that enhances sound quality with advanced equalizer and effects.

ReVanced

A continuation of the Vanced project, offering ad-free YouTube and advanced customization features.

Shamiko

A Magisk module for hiding root modifications from apps that detect them.

Riru

A Magisk module that provides an interface for other modules to work on Android without affecting system integrity.

EdXposed

A modern alternative to Xposed for Android 8.0 and above, offering a framework to run Xposed modules on newer devices.
