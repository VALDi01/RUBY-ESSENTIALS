Recovery

LineageOS Recovery by bengris

Unofficial LineageOS recovery image created by bengris32. It is very universal and used most likely in every custom rom for this phone. Unmodded, there is no ksu or magisk on it.

Pixel OS Recovery by kasun

Unofficial Pixel OS recovery image created by kasunherath. It is actually LineageOS recovery, only for Pixel OS.
USE ONLY WITH PIXELOS ROM BY KASUNHERATH.
Unmodded, there is no ksu or magisk on it.

Infinity X Recovery by Allien10x

Unofficial Infinity X or Project Infinity recovery image created by Allien10x. It is actually LineageOS recovery, only for Infinity X.
USE ONLY WITH INFINITY X ROM BY ALLIEN10X.
Unmodded, there is no ksu or magisk on it.

HyperOS recovery by Xiaomi

Official HyperOS recovery image. Basically, this is the stock recovery image that is in HyperOS. The button redirects to the xmfirmwaredownloader page, where you can download HyperOS stock images. Just download the Fastboot rom (not recovery) and extract the .tar.gz file and there should be a boot.img in the img folder.
